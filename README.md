# User Data Generator

Create .csv file with user data information using Faker.js

## Dependencies

- [Faker.js](https://github.com/Marak/Faker.js#readme)
- [Export CV](https://github.com/alexcaza/export-to-csv#readme)
- [nodemon](https://nodemon.io/)

## Installation

```bash
yarn
```

## Usage

```bash
yarn start
```
